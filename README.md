## QUICK START (2 minutes)

```bash
# 1. Unzip and enter the folder
unzip cyberspace-dashboard.zip
cd cyberspace-dashboard

# 2. Install dependencies
npm install

# 3. Run
npm run dev
```

Open → http://localhost:3000
